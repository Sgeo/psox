0x00 0x08 0x00 FNUM(1)/type FNUM(2)/port STRINGNL/addr 0x0A
    Returns: Nothing
    Creates an FD that can be read from/written to.
    Type is 0x00 for TCP, 0x01 for UDP
    Port and addr should be obvious
    
0x00 0x08 0x01 FNUM(1)/type FNUM(2)/port STRINGNL/addr 0x0A
    Returns: Nothing
    Makes a server socket. Server sockets start at 0x00 and are added at the lowest available number. Closing a server socket relinquishes that number and the other sockets in the stay at the same point.
    
0x00 0x08 0x02 FNUM(1)/serversocket 0x0A
    Returns: Nothing
    Accepts 1 connection on serversocket, making an FD
    
0x00 0x08 0x03 FNUM(1)/serversocket 0x0A
    Returns: Nothing
    Closes serversocket
